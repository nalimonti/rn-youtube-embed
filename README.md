# rn-youtube-embed

WebView based YouTube embed for React Native

## Installation

```sh
npm install rn-youtube-embed
```
or
```sh
yarn add rn-youtube-embed
```

## Usage

```js
import YoutubeEmbed from 'rn-youtube-embed';

const App = () => {
  return (
    <YoutubeEmbed
      videoId="M7lc1UVf-VE"
    />
  )
}
```
